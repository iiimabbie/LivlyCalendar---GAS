# 使用

- 使用手冊：**搜關鍵字找到該系列每池開始的日程。**<br>
    - [抽] 一般抽池。
    - [聯] 聯名抽池。
    - [福] 福袋。
    - [商] 格萊里商店 (含聯名商店)。
    - [創] 創作者商店。<br>
    
    若符合多條件，以 (聯 > 福 > 創 > 商 > 抽) 下關鍵字。<br>

- 注意事項：
    - 事件變更、新增、刪除通知一定千萬不要打開，因為有空時都在調試，你可能會吵死或信箱被塞爆。<br>
    其他活動提醒等就看個人，建議也不要因為也沒什麼料。

    - 日程時間使用JST。時程時間使用TW。<br>
    例如月初池、livly都是使用JST，因為跨日有點難決定要放在哪一天，而黑市就是使用TW時間。<br>
    已經盡量不使用到小時，但是黑市較特殊。
